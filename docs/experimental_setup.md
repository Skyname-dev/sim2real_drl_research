# Экспериментальная установка

## Симуляционная среда
- **Симулятор:** PyBullet
- **Робот:** UR5 (модель)
- **Задача:** достижение целевой точки с обходом статических препятствий

## Реальное оборудование
- **Робот:** Universal Robots UR5
- **Датчики:** встроенные энкодеры
- **Вычислительный узел:** (укажите, если есть)

## Параметры обучения DRL
| Параметр | Значение |
|:---|:---|
| Количество эпизодов | 1000 |
| Максимальная длина эпизода | 500 шагов |
| Learning rate | 3e-4 |
| Discount factor (γ) | 0.99 |
| Batch size | 64 |

## Параметры Domain Randomization
| Параметр | Диапазон вариации |
|:---|:---|
| Коэффициент трения | [0.5, 1.5] от номинала |
| Масса звеньев | [0.8, 1.2] от номинала |
| Сенсорный шум | ±5% от показаний |
| Задержка управления | [0, 0.05] секунд |

## Метрики оценки
- **Точность (Success Rate)** - процент успешных достижений цели за 100 попыток
- **Доверительный интервал** ±3% (по результатам 100 испытаний)
