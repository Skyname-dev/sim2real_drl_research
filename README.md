# sim2real_drl_research
Sim-to-Real Gap: как научить робота действовать в реальном мире, если он обучался в симуляции
## Аннотация

Данный репозиторий содержит материалы научной работы, посвящённой проблеме разрыва между симуляцией и реальностью (Sim-to-Real Gap) при внедрении глубокого обучения с подкреплением (Deep Reinforcement Learning, DRL) в управление робототехническими системами.

В работе проведён сравнительный анализ DRL-алгоритмов (DQN, PPO, SAC) и методов преодоления Sim-to-Real Gap (Domain Randomization, Domain Adaptation). Экспериментально показано, что прямое перенесение обученных политик из симуляции PyBullet на реального робота UR5 приводит к потере точности до 16%. Применение Domain Randomization сокращает разрыв до 2–3%.

## Ключевые результаты

1. **Sim-to-Real Gap** для DRL (PPO) составляет **16 п.п.** (94% в симуляции → 78% на реальном роботе)
2. **Domain Randomization** сокращает разрыв до **2 п.п.** (91% → 89%)
3. **SAC + Domain Adaptation** даёт наилучший результат: **91%** на реальном роботе
